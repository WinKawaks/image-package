# image-package

个人开发者测试多个图片加载时总苦于没有稳定的图片资源。此项目为作者从网络上获取的多张高清图片资源及其url汇总，供个人开发者测试使用。图片质量很高，普通用户也可用作壁纸。请勿用于商业用途。
